### 3. Reproducing Thesis Experiments
To reproduce any experiment or figure from the thesis, run:
```bash
python main.py --action ACTION
```
where `ACTION` is one of:
- `figure_5.x` (with `x` ranging from 1 to 8) to generate the corresponding Figure 5.x from the thesis.
- `accuracy_SGD` to compute and report the accuracies of networks trained using SGD-based optimization methods.
